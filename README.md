# SF6 Battlelog Dashboard 使い方（非エンジニア向け）

バージョン 2.5.0

---

## 事前に必要なもの

1. **Python**（推奨: 3.11 or 3.12）
   - すでに入っていればOK
   - 入っていない場合は、下記Python公式インストーラで入れてください
   - インストール時に「**Add python.exe to PATH**」にチェックを入れてください

[Python 3.12.10 インストーラ (Windows 64bit)](https://www.python.org/ftp/python/3.12.10/python-3.12.10-amd64.exe)

---

## 起動前に初回だけやること（セットアップ）

1. このフォルダを任意の場所に展開します（例: デスクトップ）
2. `install.bat` をダブルクリックします
3. 「Setup completed」と出たら完了です
   - デスクトップに「SF6 Tracker Dashboard」ショートカットが作成されます

---

## ログインが必要なとき

1. デスクトップの「SF6 Tracker Dashboard」ショートカットをダブルクリックします
2. 左のサイドバーに**プロフィールID**を入力
3. プロフィールに表示されている**プレイヤー名**を入力
4. 「**データ更新**」ボタンを押す
5. ログインが必要な場合、ブラウザが開くので **Buckler（Capcomアカウント）にログイン**
6. アプリに「**ログイン完了**」ボタンが表示されるので押す

---

## 毎回の起動方法

1. デスクトップの「SF6 Tracker Dashboard」ショートカットをダブルクリックします
   - または `run_dashboard.bat` をダブルクリックしても起動できます
2. ウィンドウが開いてダッシュボードが表示されます
   - 起動時に最新バージョンを自動チェックし、更新があれば通知されます

---

## ショートカット／タスクバーへのピン留め

- `install.bat` を実行すると「SF6 Tracker Dashboard」ショートカットが自動作成されます
  - デスクトップと、インストール先フォルダ `%LOCALAPPDATA%\SF6Tracker\` の両方に作成されます
- **タスクバーに常駐させたい場合:**
  1. ショートカット（またはアプリ）から一度起動します
  2. タスクバーに出たアイコンを右クリック →「**タスクバーにピン留めする**」を選びます
- **ショートカットを消してしまった場合:**
  - `%LOCALAPPDATA%\SF6Tracker\` を開き、中の「SF6 Tracker Dashboard」ショートカットをデスクトップにコピー、またはそこから起動してピン留めし直してください
  - 見つからない場合は `install.bat` をもう一度実行すると再作成されます（アイコンやタスクバーのまとめ表示の設定込みで正しく復元されます）

---
## データ更新方法

左のサイドバーの「**データ更新**」ボタンを押します。

---

## データ保存場所

データは以下の場所に保存されます:

```
%LOCALAPPDATA%\SF6Tracker\
```

フォルダごとバックアップしておくと安全です。

---

## うまく動かないとき

- **A)** `install.bat` をもう一度実行する
- **B)** 〈起動前に初回だけやること〉をもう一度試す
- **C)** `run_dashboard_with_prompt.bat` で起動するとエラーログが確認できます
